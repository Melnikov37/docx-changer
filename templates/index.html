📖 Простая инструкция: Как создать и заполнить документ
👋 Добро пожаловать! Эта инструкция поможет вам создать документ с автоматическим заполнением данных. Даже если вы мало работаете с компьютером - у вас всё получится!

Шаг 1 Подготовка документа в Microsoft Word

Что делать:
  1. Откройте Microsoft Word
  2. Создайте обычный документ (договор, заявление и т.д.)
  3. Вместо данных, которые будут меняться, напишите метки
⚠️ Важно! Метки должны быть в двойных фигурных скобках: {{название}}
Пример:

❌ Неправильно:

Договор с Ивановым Иваном

✅ Правильно:

{% raw %}

Договор с {{Фамилия_Имя}}

Дата: {{дата}}

Сумма: {{сумма}} руб.

{% endraw %}
💡 Как писать названия меток:
  • ✅ Можно: буквы, цифры, подчеркивания (_)
  • ❌ Нельзя: пробелы, дефисы, точки
    • {% raw %}
  • ✅ Примеры: {{Фамилия}}, {{дата_договора}}, {{сумма_рублей}}
    • {% endraw %} {% raw %}
  • ❌ Плохо: {{Фамилия Имя}}, {{дата-договора}}
    • {% endraw %}
{% raw %}📝 Совет: Используйте понятные названия меток. Например: {{телефон}}, {{адрес}}, {{email}}{% endraw %}

Шаг 2 Сохранение документа

  1. В Word нажмите "Файл" → "Сохранить как"
  2. Выберите место сохранения (например, Рабочий стол)
  3. Дайте понятное имя, например: Договор_шаблон.docx
  4. Убедитесь, что формат файла: Документ Word (.docx)
  5. Нажмите "Сохранить"
⚠️ Внимание! Файл должен быть в формате .docx, а не .doc (старый формат)

Шаг 3 Загрузка шаблона в программу

  1. На этой странице найдите синюю область с текстом "Перетащите файл .docx сюда"
  2. Способ 1 (проще):
    • Откройте папку, где сохранен ваш документ
    • Нажмите на файл левой кнопкой мыши и, не отпуская, перетащите в синюю область
    • Отпустите кнопку мыши
  3. Способ 2:
    • Нажмите на кнопку "Выберите файл"
    • Найдите ваш документ в открывшемся окне
    • Нажмите "Открыть"
✅ Готово! Программа автоматически найдет все метки в вашем документе и создаст поля для заполнения.

Шаг 4 Заполнение данных

После загрузки документа появятся поля для ввода. Просто заполните их своими данными!
Если в вашем документе были метки:

{{дата}}
{{сумма}}{% endraw %}

💡 Подсказка: Просто введите данные в каждое поле. Например, вместо {{телефон}} будет подставлено то, что вы введете.

Шаг 5 Получение готового документа

  1. Проверьте, что все поля заполнены
  2. Нажмите большую синюю кнопку "Сгенерировать документ"
  3. Подождите 2-3 секунды
  4. Появится зеленое сообщение с кнопкой "Скачать"
  5. Нажмите "Скачать" - файл сохранится на ваш компьютер
  6. Откройте скачанный файл в Word и проверьте - все метки заменятся на ваши данные!
🎉 Поздравляем! Вы создали автоматически заполненный документ!
