Как пользоваться
Добро пожаловать! Эта инструкция поможет вам создать документ с автоматическим заполнением данных.

Шаг 1 Подготовка документа в Microsoft Word

Что делать:
  1. Откройте Microsoft Word
  2. Создайте обычный документ (договор, заявление и т.д.)
  3. Вместо данных, которые будут меняться, напишите метки
{% raw %}
Важно! Метки должны быть в двойных фигурных скобках: {{название}}
{% endraw %}
Пример:

Неправильно:

Договор с Ивановым Иваном

Правильно:

{% raw %}

Договор с {{Фамилия_Имя}}

Дата: {{дата}}

Сумма: {{сумма}} руб.

{% endraw %}

Шаг 2 Загрузите шаблон и заполните данные

  1. Перетащите файл .docx на страницу или нажмите "Выбрать файл"
  2. Заполните появившиеся поля своими данными
  3. Нажмите "Сгенерировать документ"
  4. Скачайте готовый файл
